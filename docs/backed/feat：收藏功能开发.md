# 接口

### /collect接口用于点赞

1. 获取当前帖子id，调用接口获取json后返回isCollected更新哈希表（自动更新数据库）

### /collect/user接口用于查询收藏的逻辑是

1. 进入小程序后调用接口，收集接口返回的json数据并且存储在store的哈希表内。
2. 进入某个帖子后，初始化时会查询哈希表，判断用户是否收藏这个帖子，并且以此用于更新状态
